Creando un Snippet para los Media queries

1-> ctrl + shift + P 
2-> seleccionar Snippet config 
3->agregar lo siguiente
	"media query":{
		"prefix": "mq",
		"body": [
			"@include m.$1 {\n\t$2\n}" 
		]
	}

pero tambien se pueden crear atajos para agilizar el tiempo de codeo  lo anterior es a modo ejemplo per ose pueden crear muchos mas

Creando Snippet para importar Variables y  mixins
"importar mixins":{
		"prefix": "imm",
		"body": [
			"@use 'base/mixins' as m;"
		]
	},
	"importar variables":{
		"prefix": "imv",
		"body": [
			"@use 'base/variables' as v;"
		]
	}
